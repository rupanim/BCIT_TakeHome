About Parser
Welcome to the App

This is an application to convert .docx files to HTML format. Please note below the following assumptions or constraints:
  • It is a Web Application developed using AngularJS 1.8 and consumes WebAPI to parse the document.
  • Sample input file is located at '~/Contents/docs'
  • The physical location of the .docx file is the input to the application and in the output, it produces a parsed HTML in the preview as well as a physically located .html file.
  • Once the parsing is complete, preview of the HTML is available with only limited formatting.
  • The parsed HTML document is located in at the same location with the name of the HTML file same as the DocX file and images (if any) are stored in the same location under a images folder.
Thats it! To begin, click on the nav button 'Parser' on the left or scroll down.
